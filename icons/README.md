# Extension Icons

Place PNG icon files here at the following sizes:

- `16.png` — 16×16
- `32.png` — 32×32
- `48.png` — 48×48
- `128.png` — 128×128

The icons should match the Farsight brand (dark background, gold/amber accent). A simple
`F` monogram on a dark `#1a1a2e` background with `#c2a84b` color works well.

## Generating from SVG

A reference SVG is provided below. Use Inkscape, Figma, or any tool to export at the
required sizes.

```svg
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 128 128">
  <rect width="128" height="128" rx="24" fill="#1a1a2e"/>
  <text x="64" y="88" font-family="serif" font-size="80" font-weight="bold"
        text-anchor="middle" fill="#c2a84b">F</text>
</svg>
```
